# YouTube-Installer

<br>

A very simple YouTube to MP3 - MP4 - FLAC installer using yt_dlp and a beautiful CLI.

</div>
</div>

---

## Features

- YouTube -> MP3
- YouTube -> MP4
- YouTube -> FLAC

---

## Usage

Select: Option 1 for MP3 |
        Option 2 for MP4 |
        Option 3 for FLAC

Enter URL set BitRate (if MP3), enjoy
